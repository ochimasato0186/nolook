# NoLook
当アプリはAI非公開日記・感情可視化アプリNoLookです。
主にターゲットユーザーは生徒でして、生徒とAIが話していただき、悩み相談などを踏まえて、カウンセリングを行ない、そこで会話した内容を感情分析を行い学校教師に匿名で感情分析データを渡す使用になっております。

# 使用技術
フロント：Next,js
3Dモデル：Blender
バッグ：TypeScript、Python
生成AI：FastAPI、LangChain(Python)
